# Employee Analytics & Wage Gap Analysis (Machine Learning Project)

This project applies statistical modeling and machine learning techniques to analyze wage inequality and predict employee turnover using IBM SPSS Modeler.

## Objectives
- Identify wage differences based on education, experience, and gender
- Predict employee attrition using multiple ML models
- Compare model performance and detect overfitting

## Techniques Used
- Multiple Linear Regression (MLR)
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Principal Component Analysis (PCA)
- Feature Selection
- Model validation (R², p-values, Chi-square)

## Key Insights
- Education and gender were the strongest predictors of wage
- Male employees earned ~$4.50/hour more than females with similar qualifications
- Logistic regression and KNN showed strong predictive performance
- Minimal overfitting observed across models

## Tools Used
- IBM SPSS Modeler
- Excel datasets
- Statistical and machine learning techniques
